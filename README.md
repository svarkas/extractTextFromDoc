Compound Binary Document Text Extractor:
Python utility/library for extracting text content from Microsoft Compound Binary File Format (OLE2 / CFBF) documents such as .doc.

Overview:
This project implements a parser for the Compound Binary File Format (CFBF), enabling text extraction without relying on external office software.

Installation:
git clone https://github.com/svarkas/extractTextFromDoc
cd extractTextFromDoc/
pip install -r requirements.txt
